Breeze is an open-source **Remote Monitoring and Management (RMM)** platform built for MSPs and internal IT teams. It's designed to scale to 10,000+ agents while staying fast, secure, and simple to operate.

## Why Breeze?

<CardGrid>
  <Card title="Lightweight Agent" icon="rocket">
    A single Go binary (~8 MB) that runs on Linux, macOS, and Windows. No runtime dependencies.
  </Card>
  <Card title="Real-time Visibility" icon="magnifier">
    WebSocket-powered live telemetry, remote terminal, and file transfer вҖ” no polling delays.
  </Card>
  <Card title="Multi-Tenant by Design" icon="building">
    Partner вҶ’ Organization вҶ’ Site вҶ’ Device Group вҶ’ Device hierarchy, built into every query.
  </Card>
  <Card title="Security-First" icon="shield">
    SHA-256 hashed agent tokens, optional Cloudflare mTLS, RBAC, audit logging, and rate limiting out of the box.
  </Card>
</CardGrid>
