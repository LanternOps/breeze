## Deploy in Minutes

<CardGrid stagger>
  <Card title="One-Command Deploy" icon="rocket">
    `./scripts/prod/deploy.sh` brings up the full stack with auto-TLS, monitoring, and health checks.
  </Card>
  <Card title="Docker Compose" icon="seti:docker">
    Production-hardened containers with read-only rootfs, resource limits, and non-root users.
  </Card>
  <Card title="Cross-Platform Agent" icon="laptop">
    Single Go binary for Linux, macOS, and Windows. Install as a system service in one command.
  </Card>
  <Card title="Zero-Trust Ready" icon="shield">
    Optional Cloudflare mTLS client certificates, SHA-256 token hashing, RBAC, and audit logging.
  </Card>
