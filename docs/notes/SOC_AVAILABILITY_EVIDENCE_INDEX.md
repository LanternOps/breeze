## Monthly Evidence Checklist

Complete at least once per month.

- [ ] A1.1 capacity review completed and notes stored
- [ ] A1.1 alert/monitoring evidence captured
- [ ] A1.2 recovery objective changes reviewed (if any)
- [ ] A1.2 DO backup/PITR configuration evidence refreshed (or no-change attestation)
- [ ] A1.3 recovery test executed (or exception approved)
- [ ] A1.3 post-test report completed and approved
- [ ] Open corrective actions reviewed and status updated
